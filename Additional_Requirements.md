# 追加要求書

## 1. 概要
本要求書は、部分コヒーレント結像シミュレータに対して「非対称ライン・アンド・スペース（L&S）の設定機能」および「ライン・アンド・スペースの明暗逆転機能」の追加実装を定義するものです。現在の光学的計算エンジンの根幹部分を変更することなく、パターン生成プロセスおよびユーザーインターフェース（UI）を拡張し対応します。

## 2. 追加機能仕様

### 2.1 ライン・アンド・スペースの非対称設定オプション
**要件:**
ユーザーが L（ライン幅）と S（スペース幅）を個別に指定できる機能を提供する。
L と S を別々に設定できるチェックボックス「非対称 L&S (L≠S)」を用意し、チェックの状態に応じてUI・パラメータを制御する。

**UI（ユーザーインターフェース）の動作:**
1. **通常時（チェックなし）:**
   - 従来通り、L と S は同じ幅として処理する。
   - ユーザーは 1 つの「幅 (Width)」を入力する。
2. **非対称設定有効時（チェックあり）:**
   - L の入力欄と S の入力欄をそれぞれ表示する（例: L=1000nm, S=3000nm 等）。
   - その右隣に「ピッチ（Pitch）」の計算値（L + S）を自動で表示する（ユーザーは編集不可、または自動計算される表示領域とする）。
   - さらにその右隣にライン数（Lines）および方向（Ori）の入力・選択欄を配置する。

### 2.2 ライン・アンド・スペースの明暗逆転オプション
**要件:**
ユーザーが生成されるマスクの明暗（透過/遮光）を逆転できる機能を提供する。

**UI（ユーザーインターフェース）の動作:**
- 「明暗逆転 (Invert)」チェックボックスを用意する。
- **通常時（チェックなし）:**
  - ライン＝遮光 (0.0: 黒)、スペース＝透過 (1.0: 白) とする（従来通りの仕様）。
- **逆転有効時（チェックあり）:**
  - ライン＝透過 (1.0: 白)、スペース＝遮光 (0.0: 黒) に逆転させる。

## 3. 実装方針
- **UI側 (`main.py`)**: 
  - `Pattern Source` 枠内の L&S 専用フレーム (`self.frame_ls`) を再レイアウトし、新たなチェックボックスおよびテキストボックスを配置する。
  - 値変更時にピッチを自動計算しラベル・表示に反映するコールバックを設定する。プレビューも即時更新されるようにする。
- **シミュレーション・パターン生成側 (`simulation.py`)**: 
  - `generate_mask()` メソッドの引数に `space_width_nm` および `invert` フラグを追加・適用する。
  - 計算コア関数 (`simulate_image`, 光源生成部分) には一切手を加えず、生成される2次元の初期マスク配列に対してのみ形状および値（0.0/1.0）の変更を適用する。
  - `calculate_contrast()` など一部の解析ロジックに関しても、非対称パターンになった場合に適切な範囲の中央値と周辺値を取得できるように調整する。
