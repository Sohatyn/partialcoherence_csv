# Requirements Definition - Light Source Configuration

## 光源プロファイル定義

シミュレーションで使用される光源点（Source Points）の重み付けプロファイルには、以下の2つのオプションがあります。

### 1. Top-hat (一様分布)
瞳孔内の全領域で強度が一定のプロファイルです。
- **定義**: $I(r) = 1$ (ただし $0 \le r \le R_s$)
- **用途**: 標準的な部分コヒーレント照明。

### 2. Gaussian (ガウス分布)
中心から瞳孔の縁にかけて強度が減衰するプロファイルです。
- **定義**: $I(r) = \exp\left(-2 \cdot \left(\frac{r}{R_s \cdot \sigma_{gauss}}\right)^2\right)$
  - $r$: 瞳中心からの距離
  - $R_s$: 瞳半径 ($\sigma \cdot NA / \lambda$)
  - $\sigma_{gauss}$: Gaussian Sigma (1/σ) パラメータ
- **設計基準**: 
  - $\sigma_{gauss} = 1.0$ のとき、瞳の境界 ($r = R_s$) における強度は、中心強度の $1/e^2 \approx 13.5\%$ となります。
  - パラメータ値を小さくすると減衰が急激になり、大きくすると Top-hat に近づきます。
- **用途**: 実際の露光装置におけるレーザーの強度分布や、特定の照明補正を考慮したシミュレーション。

## 実装上の注意
- 積算時の総重みで正規化することで、光源形状の切り替えによる全光量の変動を防ぎます。
- 瞳孔外（$r > R_s$）の強度は常に 0 とします。
